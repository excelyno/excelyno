<p align="center">🚀 About Me</p>
Saya adalah seorang developer dengan dua "topi": sebagai Fullstack Developer, saya suka membangun aplikasi dari nol, mulai dari merancang UI yang intuitif hingga membuat backend yang andal. Di sisi lain, sebagai DevOps Enthusiast, saya terobsesi dengan otomatisasi, CI/CD, dan memastikan semua yang saya bangun berjalan mulus di cloud.

💻 Saat ini lagi asik-asiknya ngoprek Infrastructure as Code (IaC) dan orkestrasi kontainer.
